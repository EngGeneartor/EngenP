# 결제 시스템 설정 가이드

## 결제 플랫폼 비교 분석

| 플랫폼 | 카드 수수료 | 한국 지원 | 사업자 필요 | 비고 |
|--------|-----------|---------|-----------|------|
| **토스페이먼츠** | 3.3% | ✅ 네이티브 | ✅ 필요 | 카카오페이/네이버페이/토스페이 통합 |
| 카카오페이 | 3.3% | ✅ | ✅ 필요 | 카카오 생태계 한정 |
| Stripe | 3.4% + 고정비 | ⚠️ 한국 미지원 | ✅ 필요 (미국 법인) | 글로벌 서비스에 적합 |
| PayPal | 3.49% + 고정비 | ⚠️ 제한적 | ✅ 필요 | 해외 결제 위주 |

### 선정: 토스페이먼츠

- **수수료**: 카드 3.3%로 업계 최저 수준
- **결제 수단**: 카드, 카카오페이, 네이버페이, 토스페이, 계좌이체, 가상계좌 통합 지원
- **사용자 경험**: 한국 사용자에게 가장 익숙한 결제 UI
- **개발자 경험**: REST API + 웹훅 기반, 문서 잘 정리됨
- **정산**: 국내 계좌로 직접 정산 (D+1 영업일)

## 현재 구현 상태

### ✅ 코드 구현 완료

| 항목 | 파일 | 설명 |
|------|------|------|
| 토스 API 클라이언트 | `lib/toss.ts` | 결제 확인, 빌링키, 청구, 취소 |
| 결제 확인 API | `app/api/payments/confirm/route.ts` | 토스 리다이렉트 후 결제 승인 |
| 정기결제 API | `app/api/payments/billing/route.ts` | 빌링키 발급 + 자동결제 |
| 웹훅 API | `app/api/payments/webhook/route.ts` | 결제 상태 변경 수신 |
| 구독 취소 API | `app/api/payments/cancel/route.ts` | Pro → Free 다운그레이드 |
| 결제 성공 페이지 | `app/payments/success/page.tsx` | 결제 완료 후 리다이렉트 |
| 결제 실패 페이지 | `app/payments/fail/page.tsx` | 결제 실패 시 안내 |
| 구독 관리 서비스 | `lib/services/subscription.ts` | 플랜 조회, 한도 체크, 구독 갱신 |
| 업그레이드 모달 | `components/upgrade-prompt.tsx` | Pro 플랜 안내 + 결제 버튼 |
| 설정 모달 | `components/settings-modal.tsx` | 구독 상태 표시 + 해지 |

### ⏳ 연동 전 필요 사항

1. **사업자등록번호** — 토스페이먼츠 가입에 필수
2. **토스페이먼츠 API 키 발급** 후 Vercel 환경변수 설정:
   - `TOSS_PAYMENTS_SECRET_KEY` (시크릿 키)
   - `NEXT_PUBLIC_TOSS_PAYMENTS_CLIENT_KEY` (클라이언트 키)
